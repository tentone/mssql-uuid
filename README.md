# mssql-uuid
 - Go implementation of MS SQL Server Uniqueidentifier type.
 - Implements the JSON and SQL go interfaces to ensure that data is read correctly from DB.
 - Can be used alongside ORM such as (GORM)[] or (ent.io)[]
